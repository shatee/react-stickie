# react-stickie

React sticky component.  
and IE11 support!  
sticky + ie = stickie.

## Usage

```jsx
// { position: sticky; top: 10px }
return (
  <Sticky top={10}>
    <div>sticky target</div>
  </Sticky>
);
```

## example

run command

```
npm run example
```

and access to http://localhost:8080/
